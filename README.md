# Pancreas Image Classification using CNN

## Overview
This project uses a Convolutional Neural Network (CNN) to classify pancreas CT scan images. The dataset is downloaded using KaggleHub, and the model is built and trained using TensorFlow/Keras in Google Colab.

## Features
- Download dataset using KaggleHub
- Image preprocessing with ImageDataGenerator
- CNN model for image classification
- Model training and validation
- Performance evaluation
- Confusion Matrix
- Accuracy and Loss graphs

## Technologies Used
- Python
- Google Colab
- TensorFlow / Keras
- NumPy
- Matplotlib
- Scikit-learn
- KaggleHub

## Project Workflow
1. Download the pancreas CT dataset.
2. Preprocess the images.
3. Train a CNN model.
4. Evaluate model performance.
5. Display classification report and confusion matrix.
6. Plot accuracy and loss graphs.

## File
- `pancreas.ipynb` – Complete Google Colab notebook containing data preprocessing, model training, evaluation, and visualization.

## Results
The model is evaluated using:
- Accuracy
- Classification Report
- Confusion Matrix
- Training & Validation Accuracy Graph
- Training & Validation Loss Graph
